# Contributing

Thank you for helping improve `awesome-cra-compliance`.

## Scope

This repository focuses on high-value references related to the EU Cyber Resilience Act, including:

- official EU legal texts
- Official Journal references
- European Commission and EU agency material
- standards body material
- presentations and videos
- research papers and white papers
- practical compliance resources

## What To Add

Good submissions are:

- directly relevant to the CRA
- from authoritative or credible sources
- still accessible
- useful for practitioners, researchers, or policy readers
- described in one line so readers know why the link matters

Preferred source order:

1. Official EU sources
2. Standards bodies
3. Official videos or presentations
4. Research papers
5. High-quality secondary commentary

## What Not To Add

Please avoid:

- low-quality SEO articles
- duplicate links
- broken links
- promotional content with little informational value
- copyrighted files copied into the repo without permission

## Entry Format

Use this format for new links:

```md
- [Resource title](URL) - Official / Standards body / Research / Industry. One-line explanation of why it matters for CRA compliance.
```

## Style Rules

- Keep descriptions concise.
- Prefer neutral wording.
- Prefer official source links over mirrors.
- Put links in the most specific relevant section.
- If a resource is opinionated, label it clearly as commentary.

## Pull Requests

Before opening a pull request:

1. Check that the link works.
2. Check that it is in the right section.
3. Check that the description says why the resource matters.
4. Check that the formatting matches nearby entries.

## Issues

Use issues for:

- adding important missing resources
- reporting broken links
- suggesting taxonomy improvements
- flagging outdated information
