# Code Of Conduct

This repository aims to be a useful, professional, and respectful public resource.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Focus on improving the quality of the repository.
- Use evidence and sources when disagreeing on content.

## Unacceptable Behavior

- Harassment or personal attacks
- Discriminatory or abusive language
- Deliberate misinformation
- Spam or self-promotion without clear value

## Enforcement

The maintainer may remove contributions, comments, or links that do not fit the purpose of the repository or violate this code of conduct.

## Reporting

If needed, open an issue or contact the maintainer through the GitHub account profile.
